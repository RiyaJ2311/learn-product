# UX (User Experience) Audit

Audit user experience, interaction design, and usability.

## Focus Areas

1. **Navigation & Wayfinding**
   - Can users find what they need?
   - Navigation clarity, current page indication
   - Back/forward flow

2. **Interactive Elements**
   - Button affordances
   - Link styles
   - Hover/active/disabled states
   - Loading indicators

3. **User Flows**
   - Homepage → Case study
   - About → Contact
   - UX bites browsing
   - CTA journeys

4. **Friction Points**
   - Confusing interactions
   - Unclear next steps
   - Dead ends, broken links
   - Missing feedback

5. **Mobile UX**
   - Touch targets (44×44px min)
   - Scroll behavior
   - Mobile navigation
   - Gesture support

6. **Micro-interactions**
   - Hover effects, click feedback
   - Scroll animations, transitions

7. **Error Handling**
   - 404 page quality
   - Error messages
   - Recovery paths

## Deliverable

Report with:
- UX Score (0-100)
- Critical UX Issues (blocking)
- User Flow Problems
- Interaction Issues
- Mobile-specific issues
- Recommended fixes
