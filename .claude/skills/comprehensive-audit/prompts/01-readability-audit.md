# Readability & Usability Audit

Run a comprehensive readability and usability audit across ALL pages.

## Scope
- index.html, about-me.html, ux-bites.html
- All case study pages (casestudy/pages/*.html)
- UX bites detail pages (ux-bites/*.html)
- Side projects, 404.html

## Focus Areas

### 1. Typography Readability
- Line length (45-75 characters optimal)
- Line height (1.4-1.8 for body text)
- Font sizes (minimum 16px for body)
- Contrast against backgrounds
- Text hierarchy clarity

### 2. Content Scannability
- Heading structure (h1→h2→h3)
- White space usage
- Bullet points vs walls of text
- Key information prominence

### 3. Language Clarity
- Jargon usage and definitions
- Sentence complexity
- Active vs passive voice
- CTA clarity

### 4. Visual Hierarchy
- Important content standout
- Logical reading flow
- Visual weight distribution

### 5. Usability Issues
- Confusing navigation
- Unclear button labels
- Missing context
- Inconsistent patterns

## Deliverable

Create comprehensive report with:
1. Executive Summary (scores, critical issues)
2. Page-by-Page Findings
3. Priority Issues (Critical/High/Medium/Low)
4. Specific Recommendations with examples

Focus on objective, measurable issues. Provide before/after examples for fixes.
