# SEO Audit

Audit SEO implementation, meta tags, structured data, and search visibility.

## Scope
All HTML pages, robots.txt, sitemap.xml

## Focus Areas

1. **Meta Tags (Every Page)**
   - Title (50-60 chars, unique)
   - Description (150-160 chars, unique)
   - Keywords, canonical URLs
   - Robots directives

2. **Open Graph & Social**
   - og:title, og:description, og:image
   - Twitter Card tags
   - Image dimensions (1200Ã—630)

3. **Structured Data (JSON-LD)**
   - Person schema
   - Article/BlogPosting schemas
   - Review schema for testimonials
   - BreadcrumbList

4. **Technical SEO**
   - Heading hierarchy
   - Image alt text
   - Internal linking
   - URL structure

5. **Content SEO**
   - Keyword targeting
   - Content depth
   - Semantic relevance

6. **Site Configuration**
   - robots.txt rules
   - Sitemap completeness
   - 404 handling

## Deliverable

Report with:
- SEO Score (0-100)
- Missing Meta Tags (page by page)
- Structured Data Gaps
- Technical Issues
- Quick wins identified
- Expected traffic impact
