# UI Design Audit

Audit visual design quality, consistency, and design system adherence.

## Reference
Design system documented at: `.claude/rules/design-system.md`

## Focus Areas

1. **Visual Consistency**
   - Button styles, card designs
   - Typography usage
   - Spacing patterns, color usage
   - Border radius, shadows

2. **Design System Adherence**
   - CSS variable usage
   - Token compliance
   - Component patterns

3. **Visual Hierarchy**
   - Size relationships
   - Color contrast for emphasis
   - Weight distribution

4. **Spacing & Layout**
   - Consistent padding/margins
   - Alignment issues
   - Grid adherence

5. **UI Polish**
   - Rough edges, pixelated assets
   - Hover/focus/loading states

6. **Responsive Design**
   - Mobile UI quality
   - Tablet breakpoints
   - Touch target sizes

7. **Dark Mode Quality**
   - Proper implementation
   - Contrast maintenance

## Deliverable

Report with:
- UI Quality Score (0-100)
- Design Inconsistencies list
- Visual Bugs identified
- Design System Violations
- Files needing attention
- Priority fixes
