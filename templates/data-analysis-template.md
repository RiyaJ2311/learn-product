# Data Analysis Template

Use this template for any data analysis or metric review.

## Pre-Analysis Checklist

### 1. Triangulation
- [ ] What happened upstream of this metric?
- [ ] What happened downstream of this metric?
- [ ] What's the metric one level up?

### 2. Baseline
- [ ] What's the historical baseline?
- [ ] What's the target baseline?
- [ ] How does current compare?

### 3. Cohort Analysis
- [ ] What cohorts should I analyze?
- [ ] How do cohorts compare?
- [ ] Which cohorts drive changes?
