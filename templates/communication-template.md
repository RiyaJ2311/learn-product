# Communication Template

Use this template before any important communication.

---

## Pre-Communication Checklist

### 1. Answer First
- [ ] What is the EXACT question being asked?
- [ ] What's my answer in one sentence?
- [ ] Am I starting with the answer or with context?

### 2. Impact Translation
- [ ] What data am I sharing?
- [ ] What does this mean? ("This means...")
- [ ] What should we do? ("So we should...")

### 3. Self-Grade Prep
- [ ] How will I know if this landed?
- [ ] What signals will I look for?

---

## Communication Structure

### For Questions/Decisions:
```
[Answer] → [Context] → [Details if needed]
```

### For Data/Updates:
```
[Data] → "This means..." → "So we should..."
```

### For Status Updates:
```
[Status] → [Key Metrics] → [Next Steps]
```
