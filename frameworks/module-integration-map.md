# Module Integration Map

## 2-Minute Overview

The modules in this system are interconnected. Mastering one module reinforces others. This map shows how skills build on and reinforce each other, helping you see the bigger picture and create compound learning effects.

## Module Dependencies

### Prerequisites (What You Should Learn First)

| Module | Prerequisites | Why |
|--------|---------------|-----|
| 1. Communication | None | Foundation skill |
| 2. Data Storytelling | Module 1 | You need to communicate data clearly |
| 3. Ownership | Module 1 | Need clarity to make decisions independently |
| 4. Strategy | Module 1, 2 | Need to communicate and validate with data |
| 5. Leadership | Module 1, 3 | Need clarity + ownership to build trust |
| 6. PLG | Module 2, 3 | Need data skills + ownership mindset |
| 7. Thinking Patterns | Any (meta-skill) | Applies to all modules |

## Skill Combinations (Power Moves)

### Communication + Data = Persuasive Storytelling
- Answer-first + data triangulation = compelling arguments
- Impact translation + baseline context = data that drives action

### Ownership + Strategy = Execution Excellence
- Permissionless execution + working backwards = fast, aligned delivery
- Metric ownership + PRD quality = measurable outcomes

### Data + PLG = Growth Mastery
- Cohort analysis + funnel metrics = deep growth understanding
- Baseline-first + activation metrics = rigorous experimentation

### Thinking + Leadership = Strategic Trust
- Second-order thinking + impact selling = sustainable influence
- Bias awareness + stakeholder management = objective leadership

### Communication + Leadership = Executive Presence
- Answer-first + stakeholder language = executive communication
- Self-grading + trust-building = continuous improvement

## Recommended Learning Sequences

### Sequence 1: Foundation Build (8 weeks)
1. Module 1: Communication (Week 1-2)
2. Module 2: Data (Week 3-4)
3. Module 3: Ownership (Week 5-6)
4. Module 7: Thinking (Throughout)

### Sequence 2: Strategic Track (8 weeks)
1. Module 1: Communication (Week 1-2)
2. Module 4: Strategy (Week 3-4)
3. Module 5: Leadership (Week 5-6)
4. Module 7: Thinking (Throughout)

### Sequence 3: Growth Track (8 weeks)
1. Module 2: Data (Week 1-2)
2. Module 3: Ownership (Week 3-4)
3. Module 6: PLG (Week 5-6)
4. Module 7: Thinking (Throughout)

### Sequence 4: Meta-First (4 weeks intensive)
1. Module 7: Thinking (Week 1)
2. Apply thinking patterns to weakest skill module (Week 2-4)

## Daily Integration Practice

Each day, consciously connect two modules:

| Day | Connection | Practice |
|-----|------------|----------|
| Mon | Comms + Data | Present one data point with answer-first + "this means" |
| Tue | Ownership + Strategy | Make one decision without asking using strategic criteria |
| Wed | Data + PLG | Analyze one funnel metric with triangulation |
| Thu | Thinking + Leadership | Apply pre-mortem to one stakeholder interaction |
| Fri | Comms + Leadership | Review week's communications for stakeholder impact |

## The Integration Checklist

Weekly, ask yourself:

- [ ] Did I apply communication skills when sharing data?
- [ ] Did my ownership decisions align with strategy?
- [ ] Did I use data to support leadership conversations?
- [ ] Did I apply thinking frameworks to my decisions?
- [ ] Did I see connections between modules in action?
